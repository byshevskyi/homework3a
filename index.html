Linux (pronounced Listeni/ˈlɪnəks/ lin-əks[9][10] or, less frequently, /ˈlaɪnəks/ lyn-əks[10][11]) is a Unix-like and mostly POSIX-compliant[12] computer operating system (OS) assembled under the model of free and open-source software development and distribution. The defining component of Linux is the Linux kernel,[13] an operating system kernel first released on 5 October 1991 by Linus Torvalds.[14][15] The Free Software Foundation uses the name GNU/Linux to describe the operating system, which has led to some controversy.[16][17]
